# Online Movie Ticket Booking System

## Project Overview

Online Movie Ticket Booking System is a full-stack web application that allows users to browse movies, view shows, select seats, book tickets, and make payments.

## Features

### Admin

* Manage Movies
* Manage Theatres
* Manage Screens
* Manage Shows
* Activate / Deactivate Movies
* Activate / Deactivate Theatres
* Activate / Deactivate Screens
* JWT Authentication & Authorization

### User

* User Registration & Login
* Browse Available Movies
* Search Movies
* View Shows
* Seat Selection
* Ticket Booking
* Payment Processing
* Booking History
* User Profile

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* Hibernate / JPA
* MySQL

### Frontend

* React.js
* Axios
* React Router
* CSS

## Database

* MySQL

## API Security

* JWT Token Based Authentication
* Role Based Access Control (ADMIN / USER)
