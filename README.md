# 🎬 Online Movie Ticket Booking System

A Full Stack Movie Ticket Booking Web Application developed using **Spring Boot**, **React.js**, **MySQL**, and **JWT Authentication**.

---

## 🚀 Features

### 👤 User Module

* User Registration & Login
* JWT Based Authentication
* Browse Available Movies
* Search Movies by Name
* View Available Shows
* View Theatre & Screen Details
* Select Seats
* Book Movie Tickets
* Make Payments
* View Booking History
* View User Profile

### 🛠️ Admin Module

* Manage Movies
* Manage Theatres
* Manage Screens
* Manage Shows
* Activate / Deactivate Movies
* Activate / Deactivate Theatres
* Activate / Deactivate Screens
* Update Movie Details
* Monitor Bookings

---

## ⚙️ Backend Setup

1. Open backend project in Eclipse / STS

2. Configure MySQL database in:

application.properties

3. Run the Spring Boot application

---

## ⚙️ Frontend Setup

1. Open frontend project in VS Code

2. Install dependencies

npm install

3. Start React application

npm run dev
